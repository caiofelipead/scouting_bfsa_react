## PadrГөes de Projeto

### Frontend
- **Lazy Loading:** Todas as 17 pГЎginas sГЈo carregadas sob demanda via `React.lazy()`
- **Code Splitting:** Cada pГЎgina gera um chunk separado no build
- **Query Caching:** Stale time de 5-10min evita refetches desnecessГЎrios
- **Error Boundary:** Componente wrapper para captura de erros em runtime
- **Interceptors:** Token JWT adicionado automaticamente a todas as requisiГ§Гөes

### Backend
- **Service Layer:** LГіgica de negГіcio isolada em `/services/`
- **Route Layer:** Endpoints organizados em `/routes/`
- **Schema Validation:** Pydantic para validaГ§ГЈo de request/response
- **TTL Cache:** Endpoints pesados com cache de 5 minutos
- **Retry com Backoff:** Retentativas exponenciais para erros 502/503/504
